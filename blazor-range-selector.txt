# DevExpress Blazor Range Selector Documentation

> The **DevExpress Blazor Range Selector (DxRangeSelector)** allows you to display, manage, and shape tabular data.

To get started, refer to the main documentation page: [Blazor Range Selector](https://docs.devexpress.com/Blazor/405266). It provides guidance on how to navigate to the range selector class and explore documentation covering the range selector’s features and capabilities.

The class that implements the DevExpress Blazor Grid is documented here: [DevExpress DxRangeSelector component](https://docs.devexpress.com/Blazor/DevExpress.Blazor.DxRangeSelector).

The Range Selector is one of many available Blazor components. You can browse the full component list and instructions on how to build Blazor allocations based on DevExpress Blazor Components here: https://docs.devexpress.com/Blazor/llms.txt

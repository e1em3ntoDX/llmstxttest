# DevExpress Documentation

> DevExpress offers comprehensive libraries of UI controls and developer tools designed to help you build fully functional, consistent applications quickly and efficiently. These libraries support a wide range of platforms, including JavaScript (React, Angular, Vue, jQuery), Blazor, ASP.NET Core, WebForms, MVC, WinForms, WPF, VCL, and MAUI. Each control suite is accompanied by extensive documentation, including detailed API references, step-by-step tutorials, conceptual overviews, and code examples.

**Note:** DevExpress products—such as data grids, editors, and reporting tools—offer a consistent feature set across all supported platforms. To find the information you need, always specify both the platform and the product you are working with.

## Document, Report and Dashboard Modules for applications

* [Reporting](https://docs.devexpress.com/XtraReports/2162): a feature-complete reporting platform for WinForms, WPF, Blazor (Server & WebAssembly), ASP.NET Core, MVC, .NET MAUI, Angular, React, Vue and .NET 8 apps
* [Office File API](https://docs.devexpress.com/OfficeFileAPI/14911): a non-visual library that allows you to create, load, edit, export, and print rich text documents, spreadsheets, and PDF files without any UI components involved. You can also generate barcodes, compress data, and convert different measurement units.
* [Business Intelligence Dashboard](https://docs.devexpress.com/Dashboard/12049): documentation on Dashboard Designer and Dashboard Viewer that allow you to embed fully-configurable dashboards into WinForms, WPF, Blazor, Web Forms, ASP.NET Core, Angular, React, Vue apps.

## App Framework

* [XAF - Cross-Platform .NET App UI](https://docs.devexpress.com/eXpressAppFramework/112670):  a versatile application framework that allows developers to build business applications that target Windows and Web based on DevExtress UI controls and components
* [.NET App Security & Web API Service (FREE)](https://docs.devexpress.com/eXpressAppFramework/403394): creates a back-end Web API Service (Web API) with integrated authorization and CRUD operations powered by Microsoft EF Core and DevExpress XPO ORM library.
